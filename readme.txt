🎮 Mini Games Collection (Python)

This project is a collection of simple mini games built using Python, Tkinter, and Pygame. It was developed as a Grade 11 Computer Science assignment to demonstrate understanding of GUI design, game logic, and user interaction.

🧩 Games Included
🔢 High-Low Guessing Game
Try to guess the randomly generated number with hints guiding you higher or lower.
✊✋✌️ Rock Paper Scissors
Play against the computer and see if you can win the classic game.
❌⭕ Tic Tac Toe
A two-player game where players take turns marking Xs and Os on a grid.
🛠️ Built With
Python
Tkinter (for GUI and menus)
Pygame (for game rendering and interaction)
▶️ How to Run
Make sure Python 3 is installed

Install required library:

pip install pygame

(Tkinter usually comes pre-installed with Python)

Clone the repository:

git clone https://github.com/your-username/mini-games.git

Navigate into the folder:

cd mini-games

Run the program:

python main.py
